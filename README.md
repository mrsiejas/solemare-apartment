## Getting Started

1. Clone the repository
2. Set up environment variables in Cloudflare Pages:
   - `GOOGLE_MAPS_API_KEY`
   - `GOOGLE_CALENDAR_ID`
   - `ENV_FORMSPREE_ID`
   - `OPENWEATHERMAP_API_KEY` (for future use)

3. Deploy to Cloudflare Pages
